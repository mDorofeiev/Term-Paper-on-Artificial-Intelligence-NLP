# NLPSentimentAnalysis
This Jupyter notebook performs binary text classification to predict the sentiment of movie reviews. The task is to classify each text from the IMDB dataset to a category such as: Positive or Negative using feature engineering (TF-IDF vectorization with stop-words filtering) and multiple models (Multinomial Naive Bayes, Logistic Regression, and Linear SVM) to compare the results and calculate performance metrics (Accuracy, F1-Score, and time complexity) for each to identify the best fit.

Used dataset: https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews
